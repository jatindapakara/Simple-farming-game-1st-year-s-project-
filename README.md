# 🌱 EcoGamathon

EcoGamathon is a Flask-based sustainable farming simulation game developed as a 1st-year B.Tech mini project.

Players can plant crops, manage farm health, experience changing weather conditions, purchase farming resources, harvest crops, earn coins, and compete on a leaderboard while learning the importance of sustainable farming practices.

## 🚀 Features

- User login
- Crop selection and planting
- Crop growth system
- Dynamic weather effects
- Farm health management
- In-game shop
- Harvest and sell crops
- Leaderboard

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
